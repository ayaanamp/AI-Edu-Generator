# AI Educational Content Generator

## 📌 Project Overview
This project is an AI-powered study assistant that converts educational content (PDF/Text) into:
- Quiz questions (MCQ, True/False)
- Flashcards
- Audio summaries

## 🚀 Features
- Automatic quiz generation
- Flashcard creation
- Text-to-speech summaries
- Study progress tracking

## 🛠️ Tech Stack
- Python
- Streamlit
- OpenAI API
- LangChain
- SQLite
